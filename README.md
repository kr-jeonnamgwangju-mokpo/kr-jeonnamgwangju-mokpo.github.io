# 목포시 · 전남광주통합특별시

> 목포시 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-jeonnamgwangju-mokpo.github.io/**

목포시는 옛 전라남도 서남부 무안반도 남단, 영산강 하구가 바다와 만나는 자리에 앉은 항구도시다. 1897년 개항 이래 조선 4대 항구이자 한때 전국 6대 도시로 꼽히던 근대 상업항으로, 유달산 자락과 삼학도, 목포항이 어우러진 특유의 바다 풍경으로 유명하다. 남쪽 다도해로 열린 뱃길은 신안·진도로 향하는 서남해 도서 교통의 관문 구실을 해 왔고, '목포의 눈물'로 상징되는 애환의 정서와 홍어·세발낙지로 대표되는 남도 미식이 도시 정체성을 이룬다. 면적은 약 50㎢로 넓지 않지만 인구 밀도가 높은 압축 도시이며, 근대역사문화공간을 비롯한 개항기 건축 유산이 원도심에 촘촘히 남아 '살아 있는 근대 박물관'으로 불린다. 오늘날에는 해상케이블카·평화광장 춤추는바다분수 같은 관광 자산이 더해져 남도 여행의 거점으로 자리한다.
